# Console-Simulator-Game
Inspired by one of my friend's games, I recreated a typical simulator-type game about generating ores in Java. It includes:
- Luck Calculations
- Tiers
- Worlds
- Automated Rolling
- Console Input
- Admin Panel
- Save Files
- & much more

This was one of my first coding projects outside of the projects assigned to me by my highschool Computer Science class. I was relatively new to Java and did not know much, so much of the code is built inefficiently and many abstract methods were used to achieve certain features, like the Autoroll.

I had to branch out a lot to make this possible: I had no clue how to create, write, or read files, I knew nothing about threads, I had no idea how to change text color or clear the console, and I had to learn many obscure ways to get around errors and problems. I created this program to test my knowledge about Java, learn more about Java, and gain experience in solving coding roadblocks. 

This program represents more than just a simple simulator game; it depicts my first attempts at creating something unique and interesting in a coding language I barely knew. It captures a point when my skill level for programming was lower and shows how much I've grown. Most importantly, it displays my commitment & my willingness to dive deeper into programming beyond the surface of what school teaches.
