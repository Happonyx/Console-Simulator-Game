# Console-Simulator-Game
Inspired by one of my friend's games, I recreated a typical simulator-type game about generating ores in Java. It includes upgrades, a few worlds, and other special features.

This was one of my first coding projects outside of the projects assigned to me by my highschool Computer Science class. I was relatively new to Java and did not know much, so much of the code is built inefficiently and many abstract methods were used to achieve certain features, like the Autoroll.

This program represents more than just a simple simulator game; it depicts my first attempts at creating something unique and interesting in a coding language I barely knew. It captures a point when my skill level for programming was lower and represents how much I've grown.
